📥 DOWNLOAD FIREBASE SERVICE ACCOUNT KEY

The browser should have opened to Firebase Console.

If not, go to:
https://console.firebase.google.com/project/local-artist-discovery/settings/serviceaccounts/adminsdk

Steps:
1. Click "Generate New Private Key" button
2. Click "Generate Key" in the confirmation dialog
3. The JSON file will download
4. Move/rename the downloaded file to: backend/serviceAccountKey.json
5. Run: cd backend && npm run dev

Once the backend is running, phone auth will work!
